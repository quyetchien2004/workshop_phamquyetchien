Đây là phần tổng hợp các bài blog mình đã viết trong quá trình thực tập và học tập về AWS.

### [Blog 1 - Kubernetes bị sập cả Region thì sao? Đây là cách AWS Backup giúp bạn phục hồi](3.1-Blog1/)

Bài viết này tóm tắt cách dùng AWS Backup để thực hiện Cross-Region disaster recovery cho Amazon EKS, đặc biệt với các stateful workloads như MySQL và Redis. Nội dung tập trung vào luồng backup, restore, ưu điểm, hạn chế và so sánh nhanh giữa AWS Backup với Velero.

### [Blog 2 - Triển khai AWS Organizations bằng CloudFormation](3.2-Blog2/)

Bài viết này giới thiệu việc AWS Organizations đã hỗ trợ CloudFormation, giúp quản lý OU, tài khoản AWS và các chính sách quản trị theo mô hình Infrastructure as Code. Nội dung tập trung vào các resource type được hỗ trợ, ví dụ triển khai và lợi ích khi quản trị môi trường multi-account.

### [Blog 3 - Đơn giản hóa Disaster Recovery với AWS Backup, AWS DRS và Arpio](3.3-Blog3/)

Bài viết này trình bày cách kết hợp AWS Backup, AWS Elastic Disaster Recovery và Arpio để xây dựng giải pháp Disaster Recovery toàn diện hơn trên AWS. Nội dung tập trung vào bảo vệ dữ liệu, phục hồi máy chủ, tự động hóa khôi phục workload và giảm thao tác vận hành thủ công.
