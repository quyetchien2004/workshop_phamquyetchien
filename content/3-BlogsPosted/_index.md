This section summarizes the blog posts I wrote during my internship and AWS learning process.

### [Blog 1 - What if Kubernetes loses an entire Region? AWS Backup can help you recover](3.1-Blog1/)

This blog summarizes how to use AWS Backup for Cross-Region disaster recovery on Amazon EKS, especially for stateful workloads such as MySQL and Redis. It focuses on the backup and restore flow, advantages, limitations, and a quick comparison between AWS Backup and Velero.

### [Blog 2 - Deploy AWS Organizations with CloudFormation](3.2-Blog2/)

This blog introduces CloudFormation support for AWS Organizations, which helps manage OUs, AWS accounts, and governance policies using Infrastructure as Code. It focuses on supported resource types, deployment examples, and the benefits of managing multi-account environments as code.

### [Blog 3 - Simplifying Disaster Recovery with AWS Backup, AWS DRS, and Arpio](3.3-Blog3/)

This blog explains how AWS Backup, AWS Elastic Disaster Recovery, and Arpio can be combined to build a more complete Disaster Recovery solution on AWS. It focuses on data protection, server recovery, workload recovery automation, and reducing manual operation effort.
