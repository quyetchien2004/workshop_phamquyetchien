During my internship in the **First Cloud AI Journey** program from **April 17, 2026 to July 17, 2026**, I had the opportunity to learn AWS, cloud computing, and how to deploy a web application to a cloud environment.

At the beginning, my AWS foundation was still limited. Therefore, I first focused on basic concepts such as AWS Free Tier, Region, Availability Zone, IAM, EC2, S3, CloudFront, VPC, and common services used in application deployment. After that, I gradually practiced through labs, blog writing, events, and the group project **CCT Hotels Booking** on AWS.

Through the internship, I improved most in self-learning, reading technical documentation, taking work notes, and understanding how a web system is deployed outside a local environment. I also learned how to work with team members when tasks were divided into network, backend, database, frontend, SES, VNPay, and testing.

## Strengths

* I actively learned AWS from the basics because I did not have much cloud knowledge at the beginning.
* I recorded my internship progress by week, which made the report more organized.
* I participated in the **CCT Hotels Booking** project and understood the deployment flow from frontend and backend to database.
* I learned additional knowledge from AWS blogs, FCAJ events, and hands-on labs.
* I learned to view a system in layers: user access, CDN, backend, database, email, payment, and security.

## Areas for Improvement

* I need to be more proactive in asking questions when I face issues or do not fully understand a requirement.
* I need more practice in reading logs, debugging, and checking AWS issues with a clearer process.
* I need to improve team communication, especially when reporting progress or explaining technical issues.
* I need better time management to avoid pushing too much report work to the end of the period.
* I need to continue learning IAM, networking, security, monitoring, and cost optimization more deeply.

## Overall Evaluation

Overall, I evaluate my internship performance as **fair-good**. I am not yet deeply experienced in AWS, but after the internship I understand better how to learn cloud technology, how to deploy a web application on AWS, and how to document the working process in a report.

The most important lesson I gained is the mindset for learning a new technology: start from the foundation, practice step by step, check issues layer by layer, and record what I have done. This is an experience I can continue applying in future projects.
