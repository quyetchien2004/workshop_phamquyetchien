* Tìm hiểu dịch vụ lưu trữ Amazon S3.

### Kết quả đạt được tuần 4:

* Hiểu S3 là dịch vụ lưu trữ object của AWS.
* Biết bucket giống như nơi chứa dữ liệu, còn object là các file được lưu bên trong bucket.
* Biết cần kiểm tra region và quyền truy cập khi tạo hoặc dùng bucket.
* Hiểu sơ bộ việc cấu hình public access cần cẩn thận vì có thể làm lộ dữ liệu.
* Biết S3 có thể dùng để lưu file cho ứng dụng hoặc host website tĩnh trong một số trường hợp.
* Sau bài lab, em hiểu hơn các bước cơ bản khi host static website bằng S3 như tạo bucket, bật static website hosting, cấu hình index file và kiểm tra endpoint.
* Khi lên văn phòng, em có cơ hội hỏi lại những phần chưa rõ nên hiểu hơn về public access và cách kiểm tra endpoint sau khi cấu hình.
* Tuần này giúp em thấy cloud storage khác với lưu file trên máy cá nhân vì còn liên quan đến quyền truy cập và chi phí.
