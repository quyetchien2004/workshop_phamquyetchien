### Week 4 Achievements:

* I understood that S3 is an object storage service from AWS.
* I learned that a bucket is a place to store data and objects are the files inside it.
* I knew that region and access permission should be checked when creating or using a bucket.
* I understood that public access needs to be handled carefully because data may be exposed.
* I learned that S3 can be used to store application files or host a static website in some cases.
* After the lab, I understood the basic steps of hosting a static website with S3, such as creating a bucket, enabling static website hosting, setting the index file, and checking the endpoint.
* When I went to the office, I could ask about unclear parts, so I understood public access and endpoint checking better.
* This week helped me see that cloud storage is different from storing files locally because permissions and cost also matter.
